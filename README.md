# funkydoc

## Development

### CLI

#### Setup

1. Install NodeJS
2. `cd funkydoc/cli`
3. `npm ci`

#### Entrypoints

1. Parser - `cli/src/staticdocs-parser.ts`
1. Processor - `cli/src/processor.ts`

#### Running

1. cd `cli`
1. `npm i -g .`
1. `funkydoc input-graph.json output-graph.json`

### Webapp

#### Setup & Run

1. `npm install`
2. `npm start`
